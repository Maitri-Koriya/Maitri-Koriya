BeatBox :
Feel the Beat, Join the Box

BeatBox is a modern music streaming application designed for listening and downloading music through a clean and engaging mobile interface. The app focuses on smooth user experience, easy music discovery, and personalized content, making it simple for users to enjoy their favorite songs anytime.

Project information

  • Category: Mobile Application [Music Streaming ]
  • Client: Personal Project
  • Project date: 21 December, 2025
  • Design Tools: Figma (UI/UX Design, Prototyping, Design System)
  • Figma Prototype link: Open Prototype
  • Figma design link: open Design

Key Features

  • - Secure login and sign-up user interface.
  • - Home screen with artists, podcasts, playlists, albums, and trending music.
  • - Simple and clear navigation with smooth flow.
  • - Search functionality for songs, artists, albums, and podcasts.
  • - My Music section for liked songs, playlists, and downloaded content.
  • - Account screen for user profile and settings.
