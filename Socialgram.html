SocialGram:

SocialGram is a social media application inspired by Instagram, created as a learning project while exploring Figma and UI/UX design fundamentals. The app focuses on understanding real-world app structure, screen flows, and common social media features rather than advanced visual styling.

Project information

  • Category: Mobile Application UI/UX Design [Social Media]
  • Client: Personal Project
  • Project Type: UI/UX Case Study
  • Tools Used: Figma (Wireframing, UI Design, Prototyping)
  • Figma Prototype link: Open Prototype
  • Figma design link: open Design

Key Features

  • - Login and Sign-Up screens with clean form layouts
  • - Scrollable feed for viewing posts and content
  • - Search users and content easily
  • - One-to-one chat and message screens
  • - User profile with posts, followers, and following
